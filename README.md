# RandomNumberGame
A simple Java console-based Number Guessing Game where the computer generates a random number, and the player tries to guess it within a limited number of attempts.
The game provides hints after each guess, helping the player decide whether to guess a higher or lower number until the correct answer is found.
This project was created to practice core Java concepts such as:
Features
User input handling
Random number generation
Loops
Conditional statements
Input validation

The computer generates a random number between 0 and 100, and the player gets 10 attempts to guess the correct number.
